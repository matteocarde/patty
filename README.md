# Patty
The Symbolic Pattern Planner

## How to run

### Requirements
* Python 3.7
* Conda 23.7.2

### Install
1. Download this repository
2. Run ```conda env create --name patty --file=environment.yml```
3. Run ```./install```
4. Add the folder to your path

### Run
~~~
patty -o domain.pddl -f problem.pddl
~~~
